Atom使用插件精选(FE)

Posted by AlenQi on 2017-03-11

Foreword

Atom琳琅满目的插件中，很多插件良莠不齐，根据我的实践情况，为前端coder推荐一些实用的插件。

  1. sync-settings

    首先，这是一个配置备份插件，方便以后的迁移，配合gist实用快捷。

  2. atom-beautify

    使你的代码一键格式化，支持多种语言，相对较全。

  3. autoclose-html

    html代码自动化闭合，更加便捷的写div.

  4. autocomplete-paths

    更加智能的补全你的资源路径。

  5. autoprefixer

    为CSS和SCSS增加浏览器前缀。

  6. color-picker

    在编辑器选择你想要的颜色，找到颜色，点击就OK.

  7. docblockr

    辅助编写你的注释，让你爱上写注释。

  8. elastic-tabstops

    弹性制表符，适用于对于tab有强迫症的coder。

  9. emmet

    前端神器多年，前端必备，配合快捷键更个性化。

  10. file-icons

    指定文件扩展名的图标和颜色，改善视觉。

  11. git-control

    提供了一个GUI界面来管理所有常用的git命令。（如果你不喜欢命令行）

  12. gulp-control

    显示gulp任务的列表,在atom里实现GUI执行。

  13. highlight-selected

    双击时高亮当前选择的词。

  14. jquery-snippets

    支持jquery代码编写。

  15. language-vue

    高亮显示Vue代码,支持Vue组件。

  16. linter-csslint

    检测你的CSS是否基本规范，减少你的错误，提高代码质量。

  17. linter-eslint

    看你上面的，我想你也了解这个是干嘛的了。

  18. linter-sass-lint

    如果你使用SASS/SCSS，这个检测是必要的。

  19. minimap

    一个完整源代码的预览，让你看到代码地图。（不建议在一个文件里写很长的代码）

  20. minimap-highlight-selected

    在你代码概览地图上高亮显示你的选择。

  21. open-in-browsers

    你想在哪个浏览器里运行你的代码，安装这个吧。

  22. regex-railroad-diagram

    如果你经常要写正则表达式，这个插件可以显示光标下的正则表达式铁路图。

  23. simplified-chinese-menu

    Atom 的简体中文语言包，完整汉化，兼容所有已发布版本 Atom。

  24. tree-view-git-status

    树形文件视图中显示Git存储库状态。

  25. prettier-atom

    An atom package for the prettier formatter.

Tips:

  1. 之前找了很久atom的自动保存插件，没找到，原来骑驴找驴，在自带的核心扩展里面就有。autosave，把里面的配置settings里的”Enabled”勾选中，就可以在当前编辑页面失去焦点的时候自动保存。

  2. 在编辑器的核心包里有line-ending-selector可以定义你的换行类型。

  3. 这个链接是我的配置gist：
    GitHubGist（包含上面所有的插件，还有我自己的配置，可以直接导入扩展）
